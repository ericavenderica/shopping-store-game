# 🛍️ Shopping Store Game

A fun and fast-paced arcade-style shopping game! Catch the falling luxury items to earn points, avoid the explosive obstacles, and level up to unlock new treasures.

![Game Preview](/public/images/basket.png)

## 🎮 How to Play

1.  **Enter your name** to start your shopping spree.
2.  **Move your basket** to catch falling items:
    *   🛍️ **Perfume, Cap, Lipstick**: Earn points!
    *   💎 **Diamond**: Bonus points (appears in higher levels)!
3.  **Avoid obstacles**:
    *   💣 **Grenades & Bombs**: Game Over if you catch one!
4.  **Level Up**: Reach score thresholds to advance levels. The game gets faster and harder!

## 🕹️ Controls

*   **Desktop**: Use the **Left** and **Right Arrow Keys** to move.
*   **Mobile**: Tap the **Left** or **Right** side of the screen buttons.

## ✨ Features

*   **Progressive Difficulty**: 5 challenging levels with increasing speed.
*   **Dynamic Backgrounds**: Visuals change as you level up.
*   **High Score Tracking**: See your final score at the end of the run.
*   **Responsive Design**: Play on your computer or phone.
*   **Fun Animations**: Bouncy buttons, floating backgrounds, and smooth movements.

## 🛠️ Tech Stack

*   **React**: UI Library
*   **TypeScript**: Type safety
*   **Vite**: Fast build tool
*   **CSS**: Custom styling with animations

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/shopping-store-game.git
    cd shopping-store-game
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Navigate to `http://localhost:5173` to play!
